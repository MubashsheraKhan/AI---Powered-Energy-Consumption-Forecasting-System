# ⚡ AI-Powered Energy Consumption Forecasting System

## 📌 Project Overview
This project predicts household energy consumption using Artificial Intelligence.

## 🚀 Features
- Time-series forecasting
- Feature engineering (lag, time-based)
- Streamlit web app for predictions
- User CSV upload support

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## 📊 Model
- Random Forest Regressor
- Trained on Household Power Consumption dataset

## ▶️ How to Run

1. Clone repo:git clone https://github.com/AseemTapase123/Ai-powered-energy-consumption-forecasting.git


2. Install dependencies:
pip install -r requirements.txt

3. Run app:streamlit run app.py


## 📁 Input Format
Upload CSV with columns:
- Sub_metering_3
- hour
- day
- month
- weekday
- lag_1
- lag_24

## 📈 Output
- Predicted energy consumption values
